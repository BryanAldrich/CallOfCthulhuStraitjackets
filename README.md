Adds straitjackets that disable mental breaks on colonists to prevent them from hurting themselves or others.

# When equipped:



- Global Work Speed -98%
- Manipulation 0%
- Move Speed -70%
- Mental Break Threshold Boosted
- 95% chance of disabling mental break
